URP (Universal Render Pipeline):

The asset includes a sub unity package named "Universal Compatibility Package" for adding shaders compatiblity with URP (Universal Render Pipeline).
Once imported this subpackage, the shaders will no longer be compatible with LWRP (Lightweight Render Pipeline).
